Architecture Summary

Raw IoT data is generated by devices or consumer scripts and stored in MySQL. The semantic connector reads this raw data, applies SAREF-based semantic mapping, transforms it into NGSI-LD, stores processed history in MySQL, sends latest state to Eclipse Ditto, and sends NGSI-LD entities to Orion-LD.
